# OS
**Windows 94 OS** — fully functional, single-file HTML app with all requested features:

- **Apps:** Web App Maker (AI-powered via Claude API), Code App Maker (AI codegen + live preview), Notepad+++, WordPad, JS Paint (full canvas drawing), Calculator, Calendar, Pseudo-Bash Terminal, File Explorer, IPTV Playlist, Media Player, Shader Editor & Player (WebGL GLSL), JSON Editor, JS Console, Bytebeat JS (Web Audio API), Sticky Note, 2048 Game, Settings
- **Windows94 Hub:** VSCode Dev, GitHub1S, TurboWarp, Crappy FFT Vocoder, WebRAR, Web7-Zip + all 4 notation generators (Bird's Array, Bashicu Matrix, Taranovsky, Operator Notation)
- **Window functions:** drag, resize, minimize/maximize, close, z-ordering, taskbar integration
- **Settings:** 3 themes (Win98, Win7, Win10), desktop color, disk size slider (1GB–2TB with progress bar), font size
- **Desktop icons** for quick launch via double-click
